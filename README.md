# Star Warrior

## 🎮 Описание проекта

**Star Warrior** — это динамичная двухмерная аркадная игра для двух игроков, разработанная на Pygame. Игроки управляют персонажами из вселенной "Звёздных войн" — Люком Скайуокером и Дартом Вейдером, решают головоломки и проходят уровни, взаимодействуя с окружением.

### Особенности игры

- 👥 **Кооперативный геймплей** для двух игроков
- 🗺️ **5 уникальных уровней** с возрастающей сложностью
- 🧩 **Физика и взаимодействие** — ящики можно двигать, кнопки активируют механизмы
- 🚪 **Индивидуальные цели** — каждый игрок должен добраться до своей двери
- 🎨 **Анимированные спрайты** персонажей с плавной анимацией движения
- ⚡ **Прыжки и гравитация** — реалистичная физика падения

## 🎯 Управление

| Персонаж | Клавиши | Действие |
|----------|---------|----------|
| **Люк Скайуокер** | A, D | Движение влево/вправо |
| | W | Прыжок |
| | S | Движение вниз |
| **Дарт Вейдер** | ←, → | Движение влево/вправо |
| | ↑ | Прыжок |
| | ↓ | Движение вниз |

## 🚀 Установка и запуск

### Требования

- Python 3.7 или выше
- Pygame 2.1.0

### Установка зависимостей

```bash
pip install -r requirements.txt
```

или

```bash
pip install pygame==2.1.0
```

### Запуск игры

```bash
python main.py
```

## 🎲 Игровая механика

### Уровни
Игра содержит 5 предустановленных уровней. Каждый уровень представлен текстовым файлом `mapX.txt`, где:
- `s` — стартовая позиция Люка Скайуокера
- `d` — стартовая позиция Дарта Вейдера
- `h` — горизонтальная стена
- `t` — вертикальная стена
- `b` — ящик (можно двигать)
- `u` — кнопка (активирует вертикальную стену)
- `w` — дверь Люка Скайуокера (цель для 1-го игрока)
- `v` — дверь Дарта Вейдера (цель для 2-го игрока)
- `.` — пустое пространство

### Геймплей

1. **Оба игрока должны добраться до своих дверей**, чтобы победить
2. **Ящики** можно толкать, чтобы:
   - Активировать кнопки
   - Создавать мосты через препятствия
   - Блокировать опасные зоны
3. **Кнопки** активируют вертикальные стены (поднимают/опускают их)
4. **Вертикальные стены** временно открываются при нажатии на кнопку

### Физика

- Персонажи подчиняются **гравитации** — падают вниз, пока не коснутся земли
- **Прыжки** имеют параболическую траекторию
- **Коллизии** обрабатываются через маски спрайтов для точного взаимодействия
- Ящики **не проваливаются** в пустоту и могут стоять друг на друге

## 🔧 Механизмы взаимодействия

| Объект | Взаимодействие |
|--------|----------------|
| Ящик (box) | Можно толкать, активирует кнопки весом |
| Кнопка (button) | Поднимает вертикальные стены при активации |
| Вертикальная стена | Блокирует проход, пока не нажата кнопка |

## 🏆 Победа

Игра завершается, когда **оба игрока** достигают своих дверей. После этого:
- Возврат на экран выбора уровня
- Возможность выбрать следующий уровень

## 📝 Примечания

- Для корректной работы необходимо наличие всех файлов в папке `data/`
- Игра рассчитана на разрешение экрана: уровень — 1050×560, выбор уровня — 940×500
- Все графические файлы должны быть в формате `.png` или `.jpg`
- Карты уровней загружаются из файлов `data/mapX.txt`
